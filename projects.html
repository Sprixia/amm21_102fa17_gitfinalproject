A Cherokee Celebration

This is a game where the player chooses their gender, then plays through a Native American ceremony. They need to help prepare by either collecting food (women) or hunting game animals (men). The game behaves differently and changes point awards based on the player's character selection. It was made using Java and Greenfoot.

Heritage Library Foundation Website

The Heritage Library Foundation is a non-profit organization that provides resources for guests to explore their ancestry and learn more about local history in the Hilton Head region. They were previously using Textpattern for their website. It was not responsive to mobile phones or tablets and fairly out-of-date. I worked with the team assembled by the Association for Computing Machinery (ACM) chapter on campus, which I'm on the board for, to move the old website to a new and responsive content management system, SquareSpace. We used SquareSpace, Trello, and some minor custom HTML/CSS.
